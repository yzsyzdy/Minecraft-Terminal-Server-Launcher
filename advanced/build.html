    从源码构建¶

    MSTL 本身是纯 Python 脚本，直接运行即可。如果你需要打包为独立的可执行文件，以下是一些参考。

    本地开发¶

    克隆仓库¶

    git clone https://github.com/yzsyzdy/mc-server-launcher.git
cd mc-server-launcher

    直接运行¶

    python start.py

    MSTL 没有第三方依赖，不需要创建虚拟环境或安装任何包。

    代码风格¶

    项目保持相对简洁的风格，主要遵循：

    • 类型注解（typing 模块）
    • 函数和类有 docstring
    • 配置文件统一通过 config.py 读写

    打包为可执行文件¶

    使用 Nuitka（推荐）¶

    Nuitka 可以将 Python 脚本编译为独立的可执行文件，启动速度比 PyInstaller 更快。

    pip install nuitka
python -m nuitka --standalone --onefile start.py

    编译产物为 start.exe（Windows）或 start（Linux），不依赖 Python 环境。

    使用 PyInstaller¶

    pip install pyinstaller
pyinstaller --onefile start.py

    编译产物在 dist/start.exe。

    国际化贡献¶

    如果你想添加新的语言支持：

    1. 在 locales/ 目录下创建 {语言代码}.json
    2. 参考 zh.json 的键结构进行翻译
    3. 在 i18n.py 的 lang_names 字典中添加语言显示名称

    目前支持的语言：

    代码 语言 维护者
    zh 中文 yzsyzdy
    en English yzsyzdy
    ru Русский —
